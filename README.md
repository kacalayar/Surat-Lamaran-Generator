 # Generator Surat Lamaran Kerja
 
 Aplikasi web untuk membuat surat lamaran kerja secara otomatis dengan format profesional.
 
 ## Fitur
 
 - Form input data diri yang lengkap
 - Pratinjau surat secara real-time
 - Export ke PDF
 - Salin teks surat ke clipboard
 - Responsif untuk desktop dan mobile
 
 ## Tech Stack
 
 - Express.js
 - Tailwind CSS
 - html2pdf.js
 
 ## Instalasi Lokal
 
 ```bash
 npm install
 npm start
 ```
 
 Buka `http://localhost:3000`
 
 ## Deploy ke Vercel
 
 1. Push repository ke GitHub
 2. Import project di [Vercel Dashboard](https://vercel.com)
 3. Deploy otomatis
